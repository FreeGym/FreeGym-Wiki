## How to Add a New Topic

1. Create a new folder in this directory with the topic name (Title Case, spaces and ampersands allowed)
2. Add a hub markdown file inside (kebab-case filename), framed as controversy questions
3. Present competing claims fairly with multiple perspectives
4. Add evidence-needed lines pointing to specific trials, meta-analyses, or research questions
5. Open a Pull Request

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for citation standards.
